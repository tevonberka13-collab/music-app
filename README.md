# Music App 🎵

A simple app I built to track song ideas, lyrics, and concepts.

## Features
- Add song ideas
- Categories (Hook, Verse, Song Idea)
- Search
- Category filter
- Sort by newest or oldest
- Pin favorite ideas
- Export currently visible ideas as clean text
- Edit and delete ideas
- Timestamps
- Import ideas from a JSON backup
- Export ideas as a JSON backup
- Saves data locally

## Why I built this
I wanted a simple way to track my music ideas and lyrics in one place.
